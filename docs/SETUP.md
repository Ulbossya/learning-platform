# đźš€ Learning Platform - Setup Guide\n\n## Prerequisites\n\n- Java 17 or higher\n- Maven 3.8 or higher\n- Docker & Docker Compose\n- PostgreSQL 15\n- Redis 7\n- Apache Kafka\n\n## Local Development Setup\n\n### 1. Clone Repository\n\n```bash\ngit clone https://github.com/Ulbossya/learning-platform.git\ncd learning-platform\n```\n\n### 2. Start Infrastructure\n\n```bash\ncd infra/docker\ndocker-compose up -d\n```\n\nThis starts:\n- PostgreSQL (port 5432)\n- Redis (port 6379)\n- Kafka (port 9092)\n- Prometheus (port 9090)\n- Grafana (port 3000)\n- Elasticsearch (port 9200)\n- Kibana (port 5601)\n\n### 3. Build All Services\n\n```bash\nmvn clean package -DskipTests\n```\n\n### 4. Run Services\n\n#### Option A: Run individually\n\n```bash\n# Auth Service (port 8081)\ncd backend/auth-service\nmvn spring-boot:run\n\n# User Service (port 8082)\ncd backend/user-service\nmvn spring-boot:run\n\n# Course Service (port 8083)\ncd backend/course-service\nmvn spring-boot:run\n```\n\n#### Option B: Run all with Maven\n\n```bash\nmvn spring-boot:run -pl api-gateway\n```\n\n### 5. Access Services\n\n- **Auth Service**: http://localhost:8081/swagger-ui.html\n- **User Service**: http://localhost:8082/swagger-ui.html\n- **Course Service**: http://localhost:8083/swagger-ui.html\n- **Prometheus**: http://localhost:9090\n- **Grafana**: http://localhost:3000 (admin/admin)\n- **Kibana**: http://localhost:5601\n\n## Testing\n\n### Run All Tests\n\n```bash\nmvn test\n```\n\n### Run Integration Tests\n\n```bash\nmvn verify\n```\n\n### Generate Coverage Report\n\n```bash\nmvn clean test jacoco:report\n```\n\nView coverage: `target/site/jacoco/index.html`\n\n## Database\n\n### Initialize Database\n\nFlyway migrations run automatically on service startup.\n\n### Manual Database Connection\n\n```bash\npsql -h localhost -U postgres -d learning_platform\n```\n\n## API Testing\n\n### Using Swagger UI\n\nEach service has Swagger UI at `/swagger-ui.html`\n\n### Using cURL\n\n```bash\n# Register user\ncurl -X POST http://localhost:8081/api/v1/auth/register \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\n    \"email\": \"user@example.com\",\n    \"password\": \"SecurePass123!\",\n    \"fullName\": \"John Doe\",\n    \"role\": \"STUDENT\"\n  }'\n\n# Login\ncurl -X POST http://localhost:8081/api/v1/auth/login \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\n    \"email\": \"user@example.com\",\n    \"password\": \"SecurePass123!\"\n  }'\n```\n\n## Docker Commands\n\n```bash\n# View running containers\ndocker-compose ps\n\n# View logs\ndocker-compose logs -f\n\n# Stop services\ndocker-compose down\n\n# Remove all data\ndocker-compose down -v\n```\n\n## Troubleshooting\n\n### Port Already in Use\n\n```bash\n# Find process using port\nlsof -i :8081\n\n# Kill process\nkill -9 <PID>\n```\n\n### Database Connection Issues\n\n```bash\n# Check PostgreSQL\ndocker-compose logs postgres\n\n# Check connection\npsql -h localhost -U postgres -c \"SELECT 1\"\n```\n\n### Redis Connection Issues\n\n```bash\n# Test Redis\nredis-cli ping\n```\n\n## Environment Variables\n\nCreate `.env` file:\n\n```\nJWT_SECRET=your-secret-key-here\nJWT_EXPIRATION=86400000\nDB_HOST=localhost\nDB_PORT=5432\nDB_NAME=learning_platform\nDB_USER=postgres\nDB_PASSWORD=postgres\nREDIS_HOST=localhost\nREDIS_PORT=6379\nKAFKA_BOOTSTRAP=localhost:9092\n```\n\n##
